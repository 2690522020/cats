  6. def catchOnly[T >: Null <: Throwable]: CatchOnlyPartiallyApplied[T]

    Evaluates the specified block, catching exceptions of the specified type and returning them on the left side of the resulting Xor.

    Evaluates the specified block, catching exceptions of the specified type and returning them on the left side of the resulting Xor. Uncaught exceptions are propagated.

    For example:

    scala> Xor.catchOnly[NumberFormatException] { "foo".toInt }
res0: Xor[NumberFormatException, Int] = Left(java.lang.NumberFormatException: For input string: "foo")

    This method and its usage of NotNull are inspired by and derived from the fromTryCatchThrowable method contributed to Scalaz by Brian McKenna.

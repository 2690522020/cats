ReaderWriterStateT 

Companion object ReaderWriterStateT

final class ReaderWriterStateT[F[_], E, L, S, A] extends Serializable

Represents a stateful computation in a context F[_], over state S, with an initial environment E, an accumulated log L and a result A.

In other words, it is a pre-baked stack of ReaderT[F, E, A], WriterT[F, L, A] and StateT[F, S, A].
